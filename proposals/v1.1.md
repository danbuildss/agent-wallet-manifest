# Proposal: v1.1

**Status:** Draft — not yet open for community discussion

This proposal is intentionally deferred until v1 has meaningful adoption.

When ready, this proposal will be converted to a GitHub Issue for community discussion.

---

## Planned scope

- Optional wallet-level `proof` stub (for future attestations and signatures)
- Optional root-level `tags` (for ecosystem categorization)
- Formal deprecation plan for `x` → `x_handle` and `fee_recipient` → `fee_receiver`
- `$id` alignment

## Tracking

All items are non-breaking. No existing v1 manifests will be invalidated.

Opening this as a GitHub Issue is the trigger to begin the community discussion period.
